# RN-E2E-Bitrise-sample

## Environment

- React Native 0.63
- Detox 17.14.3
- macOS 10.15

## Install

```shell
yarn
```

## Android

### Detox Build

```shell
yarn build:android:ci
```

### Creating an Android Emulator

```shell
yarn android:setup
```

### Detox Test

```shell
yarn test:android:ci
```
